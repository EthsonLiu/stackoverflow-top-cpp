## 问题

什么是段错误？在 C 和 C++ 中有区别么？段错误和空悬指针有关系么？

## 回答

段错误是由于程序访问了本不属于它的的内存而引起的错误。

每当遇到段错误时，你就应该知道程序在内存访问上出错了。比如，访问了已释放的变量、写入只读内存......在大多数语言中，段错误在本质上都是相同的，在 C 和 C++ 中也是一样。

要想重现段错误很简单，解引用一个空指针就会出现，

```c++
int *p = NULL;
*p = 1;
```
